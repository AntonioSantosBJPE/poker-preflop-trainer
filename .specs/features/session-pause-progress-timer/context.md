# Contexto: Decisoes do Utilizador

## Sessao de discussao (2026-05-03)

### Visual da pausa

- **Overlay sutil + texto "Pausada"**: manter a mao visivel mas com overlay semitransparente e indicador de pausa.
- Nao esconder cartas.

### Pausa simultanea

- **Global**: um unico botao Pausar/Continuar no cabecalho geral pausa/retoma todas as mesas juntas.

### Barra de progresso

- **Barra horizontal fina**: `<progress>` ou div estilizada logo abaixo do texto "Mao X / Y".

### Icone do timer

- **Timer (⏱)**: `TimerIcon` do Lucide React.
